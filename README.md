# Boutique Authority Presence System

A premium Next.js landing website for boutiques, fashion stores, modest fashion shops, luxury retail sellers, clothing brands, and accessories businesses.

This version includes:

- editorial luxury hero
- mobile-refined lookbook layout
- premium product catalogue cards
- WhatsApp and call CTAs
- customer trust badges
- “How ordering works” section
- size / payment / delivery guidance panels
- concierge inquiry form UI
- public-facing retail copy
- premium footer with navigation and contact placeholders
- SEO and Open Graph metadata

## Edit first

Most boutique content is in:

```txt
lib/site-data.ts
```

Edit collection names, product cards, prices, sizes, reviews, trust badges, order steps, and footer links there.

## Run locally

```bash
npm install --registry=https://registry.npmjs.org/ --no-audit --no-fund
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Build

```bash
npm run build
```

## Replace placeholders before selling

Update:

- WhatsApp number
- phone number
- email address
- boutique name
- collection names
- product prices
- size guide details
- delivery/payment details
- real customer reviews
- real product photos when available

No backend is included. The inquiry form is front-end UI only.
